# Agent 执行索引

本文件用于初始化本项目的 AI 协作上下文。每次执行需求、修复、重构、联调、评审或生成文档前，必须先阅读并遵守下列 3 个 Markdown 文件。

## 必读文档

1. `docs/权限安全的企业 RAG 智能知识库平台产品实施方案.md`
   - 用途：理解产品目标、功能范围、技术架构、RAG 主链路、实施顺序、测试方案、风险和回滚要求。
   - 使用时机：新增业务模块、调整 RAG 链路、实现文档/索引/检索/问答/评测/运维能力前必须阅读。

2. `docs/后端权限逻辑补充及数据库表结构完善.md`
   - 用途：理解前后端权限契约、认证接口、系统权限、资源权限、错误码、审计要求和权限相关数据库设计。
   - 使用时机：实现或修改登录、当前用户、角色权限、空间 ACL、文档 ACL、接口鉴权、审计日志前必须阅读。

3. `docs/development/engineering-standards.md`
   - 用途：统一本仓库编码规范、模块边界、API 规范、数据库规范、RAG 安全规范、测试门禁、提交规范和 AI 协作要求。
   - 使用时机：所有代码变更、SQL 变更、配置变更、接口变更和交付说明前必须阅读。

## 执行规则

- 先确认当前任务涉及的业务域，再回到对应必读文档核对约束。
- 涉及 API、数据库、权限、安全、配置、索引、模型、审计、回滚的变更，必须在交付说明中显式列出验证结果和未验证风险。
- 不得信任前端传入的 `tenantId`、`userId`、`ownerId`、`createdBy` 等服务端字段，必须以后端认证上下文为准。
- 不得提交真实密码、token、API Key、公网数据库账号或私有部署凭证。
- 不得回退用户已有改动，不得删除无关未跟踪文件。

## 项目基线

- 后端根包：`com.meishuosoft.rag`
- 构建工具：Maven
- Java 版本：17
- 框架：Spring Boot 3.3.5
- 数据库：MySQL 8
- 缓存：Redis 7
- 检索：Elasticsearch 8
- 文档解析：Apache Tika
- 文件存储：MinIO 或本地文件存储

## 常用验证命令

```bash
mvn test
mvn clean package
docker compose up -d mysql redis elasticsearch minio
```

无法执行上述命令时，必须说明原因和替代验证方式。
